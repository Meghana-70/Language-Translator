# Language Translator

## Project Overview

Language Translator is a web-based application developed using Python, Flask, HTML, and CSS. It enables users to translate text between multiple languages through a simple and interactive interface.

## Features

* Translate text between multiple languages
* User-friendly web interface
* Supports real-time translation
* Fast and accurate language conversion
* Simple and responsive design

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Deep Translator Library

## How to Run

1. Install the required libraries:
   pip install flask deep-translator

2. Navigate to the project folder.

3. Run:
   python app.py

4. Open the URL displayed in the terminal (usually http://127.0.0.1:5000).

5. Enter text, choose a target language, and click **Translate**.

## Supported Languages

* Telugu
* Hindi
* French
* Spanish

## Project Workflow

1. User enters text.
2. User selects a target language.
3. The application sends the text for translation.
4. The translated text is displayed on the screen.
